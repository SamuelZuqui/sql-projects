Projeto de Banco de Dados: HospitalDB
Este é um projeto simples para a disciplina de Banco de Dados, focado em criar um pequeno esquema para um hospital.

Sobre o Projeto
O repositório contém scripts SQL para criar, popular e consultar um banco de dados fictício de um hospital.

A pasta HospitalDB contém os arquivos .sql que executam as seguintes ações:

Criação do Banco (DDL): Define a estrutura das tabelas Medico, Paciente e Atendimento, incluindo chaves primárias e estrangeiras.

Inserção de Dados (DML): Popula as tabelas com dados fictícios para médicos, pacientes e seus atendimentos.

Consultas (SELECT): Inclui vários exemplos de consultas para extrair informações do banco, demonstrando o uso de JOIN, GROUP BY e funções como COUNT e SUM.

O objetivo é demonstrar os conceitos básicos de modelagem e consulta em SQL.
